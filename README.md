# 🚀 Job Search Agent

## 📌 Overview

Job Search Agent is an AI-powered application that helps users find relevant job opportunities based on their skills, experience, and preferences. The system intelligently searches and filters jobs to provide personalized recommendations, saving users time and effort in their job search process.

---

## ✨ Features

- 🔍 Automated Job Search
- 🎯 Skill-Based Job Matching
- 📍 Location-Based Filtering
- 🤖 AI-Powered Recommendations
- 📄 Job Description Analysis
- ⚡ Fast and User-Friendly Interface
- 📊 Organized Job Listings

## 🏗️ Project Architecture

1. User enters skills, job role, and preferences.
2. The agent processes user requirements.
3. AI analyzes the input.
4. Relevant jobs are searched and filtered.
5. Personalized recommendations are generated.
6. Results are displayed through an interactive interface.

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/job-search-agent.git
```

Move into the project folder:

```bash
cd job-search-agent
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
